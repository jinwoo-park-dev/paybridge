Project Docs

PayBridge is a Java 21 and Spring Boot payment orchestration backend that keeps one shared payment and reversal lifecycle across Stripe and NicePay. The public EC2 demo keeps runtime Swagger UI disabled; this site publishes the JSON API reference from the repository.

API Reference Swagger UI rendered from the OpenAPI contract in this repository. Try it out is disabled. OpenAPI YAML Source contract for PayBridge JSON endpoints. Architecture Decisions ADR files for provider boundaries, outbox tradeoffs, and server rendered operator UI. Source Repository Spring Boot service, tests, and deployment scripts.

Operator APIs remain protected by authentication in runtime environments and can be disabled by feature flag.
