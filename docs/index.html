Project Docs

Version-controlled documentation for the PayBridge portfolio project. The public EC2 demo intentionally keeps runtime Swagger UI disabled, while this GitHub Pages site exposes the API reference for recruiter and interviewer review.

API Reference Read-only Swagger UI generated from the checked-in OpenAPI contract. OpenAPI YAML Source contract for PayBridge JSON endpoints. Architecture Decisions ADR files for provider boundaries, outbox tradeoffs, and server-rendered operator UI. Source Repository GitHub repository with the Spring Boot service, tests, and deployment scripts.

The API reference is intentionally read-only. Operator APIs remain protected by authentication in runtime environments and can be disabled in the production profile.
